Libero Credits

  • Leif Svalgaard wrote the original ETK Dialog Designer that generated COBOL code. Libero is directly descended from this tool.
  • Libero was written by Pieter Hintjens.
  • Christian Rozet proposed the concept of a multithreading FSM.
  • Claude Phillipo and his colleagues beta-tested Libero for Windows.
  • David Milner (ex-Streetlife) provided the music.
  • Inge Heyvaert kept the music going.
  • Ewen McNeill provided the OS/2 binary and source changes for OS/2.
  • John Dammeyer and Venu Nayar helped debug the Windows version.
  • Glenn R. Williams and Joe McVerry beta-tested the Rexx version.
  • Neil Bowers wrote weblint, the tool I use to check the web pages.
  • And many, many people provided helpful comments, criticism, and advice for which we are grateful. Keep the mail coming!
